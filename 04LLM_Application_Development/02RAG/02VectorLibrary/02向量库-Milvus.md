Milvus 常用的底层检索机制及核心原理：

| 索引/检索机制                               | 简要说明                                             |
| ------------------------------------------- | ---------------------------------------------------- |
| **FLAT（暴力搜索）**                        | 不构建索引，针对较小数据集执行全表扫描，获取精确结果 |
| **IVF 系列（IVF_FLAT / IVF_SQ8 / IVF_PQ）** | 倒排文件索引，向量被分簇，先缩小搜索范围再搜索       |
| **HNSW（图索引）**                          | 图结构导航搜索，适合低延迟高召回                     |
| **HNSW_PQ（混合压缩图）**                   | HNSW + Product Quantization 压缩提高内存效率         |
| **DISKANN（磁盘近邻）**                     | 支持磁盘数据扩展大规模向量检索                       |
| **Hybrid Search**                           | 稠密向量 + 稀疏向量/关键词组合增强相关性             |

总结对比：

| 类型     | 搜索速度 | 精确度        | 内存 | 适用         |
| -------- | -------- | ------------- | ---- | ------------ |
| FLAT     | 慢       | 高            | 低   | 小规模       |
| IVF 系列 | 快       | 中等↑         | 中等 | 大数据       |
| HNSW     | 很快     | 高            | 高   | 实时大数据   |
| HNSW_PQ  | 快       | 中等↑         | 低   | 内存受限     |
| DISKANN  | 中等     | 中等          | 低   | TB 级数据    |
| Hybrid   | 取决组合 | 强语义+关键词 | 不一 | 复杂检索场景 |
